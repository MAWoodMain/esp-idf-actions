Web flasher demo

This is a bare minimum example of a github esp-idf project which auto builds using github actions and offers a github pages website with esp-web-flasher built-in

This exampling is building the ESP-IDF V4.4 hello world example for the ESP32-C3 platform, but it can be trivially changed to support any target supported by the ESP-IDF, see the main.yml file in the github source for configuration

The demo is not available because your browser does not support Web Serial. Open this page in Google Chrome or Microsoft Edge instead (but not on your iOS device).
